# # 数据类型介绍

即在自定义函数中支持输入的数据的类型，包括：

  1. Integer - 整型类型，值只能为整数

  2. String - 字符串类型，使用时用""封装

  3. BigDecimal - 数字类型

  4. Boolean - 布尔类型，值只有true和false

  5. Date - 日期类型，格式为 YYYY-MM-DD，使用时用""封装

  6. Time - 时间类型，格式为 hh:mm，使用时用""封装

  7. DateTime - 日期类型（具体），格式为 YYYY-MM-DD hh:mm，使用时用""封装

  8. List - 集合类型，使用时用[ ]封装，中间的数据用,分隔

  9. Map - 集合类型，使用时用[ ]封装，中间的数据用,分隔，中间的数据为key:value键值对的形式存在

  10. Duration- 无创建接口，会在Integer类型数据调用日期类型数据时返回

  11. QueryResult - 无创建接口，会在对象查询接口中返回

  12. HttpResult - 无创建接口，会在httpAPI接口中返回

  13. Range - 用于循环逻辑
