# # QueryResult类型

QueryResult - 无创建接口，会在对象查询接口中返回

属性：
    
    
    size：查询的数据结果的条数
    
    total：总数据条数
    
    dataList：查询返回的数据列表
    
    

详细用法见：Fx.object.find
