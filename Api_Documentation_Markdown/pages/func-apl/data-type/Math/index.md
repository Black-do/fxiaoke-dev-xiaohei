#  Math类型

Math类型 Math - 数学计算相关的方法

Math类型的静态方法：

  * BigDecimal.sqrt: 开平方
  * BigDecimal.abs: 取绝对值
  * BigDecimal.max: 取两者中更大的
  * BigDecimal.min: 取两者中更小的
  * BigDecimal.ceil: 向上取整
  * BigDecimal.floor: 向下取整
  * BigDecimal.round: 四舍五入



例：
    
    
    log.info(Math.sqrt(16)) //开平方
    log.info(Math.abs(-1)) //取绝对值
    log.info(Math.max(10,16)) //取两者中更大的
    log.info(Math.min(10,16)) //取两者中更小的
    log.info(Math.ceil(7.5 as double)) //向上取整
    log.info(Math.floor(7.9 as double)) //向下取整
    log.info(Math.round(7.5 as double)) //四舍五入
