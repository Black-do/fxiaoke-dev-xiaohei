# # HttpResult类型

HttpResult - 无创建接口，会在httpAPI接口中返回

属性：
    
    
    statusCode：HTTP响应状态码
    
    content：HTTP响应的Payload内容
    
    header：HTTP响应的header内容
    

详细用法见：Fx.http.get/post
